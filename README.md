# 404-to-mars

"404 error" is a group project for a competition about creating a Mars-inspired error page.

## Authors and Acknowledgements

Thank you to our advanced web design teacher for encouraging us to participate in this contest. Working as a group made the process a lot more appealing and allowed us to achieve nice results in less time. Although it was frustrating at times, we made it through together, and learned from the different obstacles we faced along the way. Thank you to our group, Aimar, Ana, Diana, Ruimin, and Izaro, developers and survivors of this project.

## Purpose and Audience

    -**For Students:** A challenge to encourage creativity and problem resolution in groups.
    -**For Professors:** A way to make the class inspired and keep them motivated.

    -**Purpose:** To create a single error 404 page that is inspired by mars and space using different animation and creative techniques learned in class. The main goal was to make it appealing, responsive and interactive for the user.

## Workflow Breakdown

1.  Creating the new repository and sharing it with our fellow group members and teacher.
2.  Division of the different tasks.
3.  Creation of wireframes and discussion of ideas.
4.  Illustration of the different elements needed.
5.  Creating the main code with Html, CSS, Javascript and different asset and icon folders.
6.  Adding a CSS parallax scroll to the elements.
7.  Adding music.
8.  Adding texts and interactive buttons.
9.  Make it responsive.
10. Making sure the web was working properly across various devices and browsers.
